Configuration Help

Getting Started:

  1. Ensure your fine-tuned model is in the "checkpoint-2200" directory
  2. Install required dependencies: pip install torch transformers accelerate
  3. Select your model above and save the configuration
  4. Start chatting with your fine-tuned Mixtral model!

Model Options:

  • Local Models: Your fine-tuned Mixtral models (checkpoint-2200, checkpoint-3000)
  • DeepSeek API: External DeepSeek models for chat and coding
  • OpenAI API: GPT models for general conversation
  • Anthropic Claude: Claude models for advanced reasoning

Parameters:

  • Max Tokens: Controls response length (higher = longer responses)
  • Temperature: Controls creativity (higher = more creative, lower = more focused)
